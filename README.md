# LAB-001 – HomeLab

## Objectif
Créer un environnement de test permettant de reproduire des incidents systèmes et réseaux courants en entreprise.

## Infrastructure

### Systèmes
- Windows 10 / 11 (poste client)
- Windows Server (Active Directory)
- Linux (optionnel)

## Réseau
- Routeur (simulé ou pphysique)
- Switch (Cisco Packet Tracer ou réel)
- DHCP
- DNS

## Topologie simplifiée
Internet
⬇ 
Routeur
⬇ 
Switch
⬇ 
├── Poste client Windows
└── Serveur Windows (AD / DNS / DHCP)

## Utilisation du Lab

Ce laboratoire permet de simuler :

- Perte d'accès Internetù
- Problème DNS
- DHCP défaillant
- Poste hors domaine
- Problèmes de connectivité réseau

## Compétences travaillées

- TCP/IP
- Architecture réseau
- Active Directory
- DNS / DHCP
- Diagnostic d'incidents

## Objectif pédagogique

Développer une logique de technicien support / systèmes en reproduisant des incidents réels rencontrés en entreprise.
